页面交互都是通过设置控件的href或者onclick属性的地址来进行的。
href和onclick支持的地址类型都是相同的，这些不同的类型各有特征：
静态页面地址：以res:开头的地址，如res：page/index.xhtml
动态页面地址：以http://开头的地址，如：http://www.baidu.com
执行内置函数：以script:开头的地址，如：script:close
执行JS脚本：通常是一段JS代码，如：alert('此href触发的是一个JS函数')
只要具有href或者onclick属性的控件都可以使用上面四种方式进行页面的交互。除了a控件外，div、grid、input:button、img、menubar、tab等等控件都具有href或者onclick属性
